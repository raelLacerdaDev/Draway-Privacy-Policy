
# Draway - Política de Privacidade

Este repositório contém a página web estática da Política de Privacidade do aplicativo **Draway**. 

Esta página foi desenvolvida para ser hospedada via [GitHub Pages](https://pages.github.com/) e tem como objetivo principal atender aos requisitos de conformidade da Apple App Store (App Store Connect) referentes a políticas de privacidade de aplicativos.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Utilizando tags semânticas (`<main>`, `<article>`) para melhor acessibilidade, estruturação e sem o uso de `<div>`.
* **CSS3:** Design responsivo e limpo através de um arquivo `.css` separado, sem a utilização de frameworks externos, garantindo um carregamento rápido.

## 📱 Sobre o Aplicativo e a Política
O Draway é um aplicativo construído sob o princípio de privacidade desde o design (*privacy by design*). O aplicativo funciona de maneira totalmente offline, não coletando, transmitindo ou compartilhando qualquer dado pessoal ou de uso com servidores externos ou terceiros. Todo conteúdo criado é armazenado exclusivamente no armazenamento local do dispositivo do próprio usuário.
